## 🚀 البدء

### 1. فك الضغط
```bash
unzip esa-hub-professional.zip
cd esa-hub-professional
```

### 2. التشغيل
```bash
python3 -m http.server 8000
```

ثم اذهب إلى: `http://localhost:8000`
