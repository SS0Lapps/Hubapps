# Esahub - منصة المشاريع الاحترافية

## 📋 نظرة عامة

Esahub هو موقع ويب احترافي مصمم لعرض المشاريع والخدمات الرقمية. يتضمن نظام Developer Mode متقدم لتتبع الزوار وإدارة المحتوى.

## ✨ الميزات الرئيسية

### 1. واجهة مستخدم احترافية
- تصميم حديث وأنيق
- متجاوب تماماً (Responsive)
- تأثيرات حركية سلسة
- دعم اللغة العربية (RTL)

### 2. Developer Mode - لوحة التحكم الخفية
- **الوصول**: اضغط `Ctrl+Shift+D` أو انقر على "اضغط Ctrl+Shift+D" في التذييل
- **ميزات**:
  - تتبع بيانات الزوار (IP، الدولة، المدينة، ISP، المتصفح)
  - سجل الزوار المفصل
  - إحصائيات شاملة
  - تعديل المحتوى الديناميكي
  - إضافة مشاريع جديدة
  - تشغيل كود JavaScript مخصص

### 3. نظام إدارة المحتوى
- تعديل عنوان الموقع
- إضافة/حذف المشاريع
- تخزين البيانات محلياً (localStorage)
- نسخ احتياطية تلقائية

### 4. نظام التتبع المتقدم
- تتبع الزيارات والنقرات
- تتبع التمرير والتفاعلات
- تحليل السلوك
- تقارير مفصلة

### 5. نظام Admin Panel
- إدارة الإعدادات
- إضافة CSS مخصص
- إضافة JavaScript مخصص
- إدارة السجلات
- النسخ الاحتياطية والاستعادة

## 🚀 البدء السريع

### 1. فتح الموقع
افتح ملف `index.html` في متصفح الويب

### 2. الوصول إلى Developer Mode
- اضغط `Ctrl+Shift+D` على لوحة المفاتيح
- أو انقر على النص "اضغط Ctrl+Shift+D" في أسفل الصفحة

### 3. استكشاف الميزات
- **تبويب الزوار**: شاهد معلومات الزائر الحالي وسجل الزوار
- **تبويب التحكم**: عدّل المحتوى وأضف مشاريع جديدة
- **تبويب الإحصائيات**: شاهد الإحصائيات الشاملة

## 🔧 الدوال الرئيسية

### إدارة البيانات
```javascript
// الحصول على البيانات
getStorageData('key')

// حفظ البيانات
setStorageData('key', value)

// حذف البيانات
removeStorageData('key')

// حذف جميع البيانات
clearAllStorageData()

// تصدير البيانات
exportAllData()
```

### التتبع
```javascript
// تتبع حدث مخصص
trackEvent('eventName', { data })

// الحصول على الأحداث
getTrackedEvents()

// الحصول على ملخص التحليلات
getAnalyticsSummary()
```

### Admin Panel
```javascript
// تسجيل الدخول
adminLogin('password')

// تسجيل الخروج
adminLogout()

// إضافة سكريبت مخصص
addCustomScript('code', 'name')

// تحديث الإعدادات
updateSiteSettings({ setting: value })
```

## 🔐 الأمان

**ملاحظة مهمة**: هذا الموقع مصمم لأغراض تعليمية وتوضيحية فقط. لا يجب استخدامه في بيئة إنتاجية بدون تحسينات أمنية.

- كلمة مرور Admin بسيطة: `admin123`
- البيانات تُخزن محلياً في localStorage
- لا يوجد تشفير للبيانات الحساسة
