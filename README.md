# Portfolio HUB - جميع المشاريع في مكان واحد

مركز تجميع احترافي لجميع المشاريع مع واجهة جميلة وسهلة الاستخدام.

## ✨ المميزات

- ✅ تصميم احترافي وعصري
- ✅ صفحة رئيسية جميلة
- ✅ قسم Shortcuts جاهز للمشاريع
- ✅ زر TikTok بارز
- ✅ أكواد نظيفة وسهلة التعديل
- ✅ متجاوب مع جميع الأجهزة

## 🚀 البدء

### 1. فك الضغط
```bash
unzip portfolio-hub.zip
cd portfolio-hub
```

### 2. التشغيل
```bash
python3 -m http.server 8000
```

ثم اذهب إلى: `http://localhost:8000`

## 📝 إضافة مشاريع

### الطريقة 1: عبر JavaScript Console
```javascript
addProjectShortcut('اسم المشروع', 'وصف قصير', 'https://رابط-المشروع', '🎯');
```

### الطريقة 2: في ملف shortcuts.js
```javascript
addProjectShortcut('اسم المشروع', 'وصف قصير', 'https://رابط-المشروع', '🎯');
addProjectShortcut('مشروع آخر', 'وصف آخر', 'https://رابط-آخر', '💻');
```
